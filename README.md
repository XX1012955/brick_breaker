# 🧱 美化版打砖块游戏

这是一个用 Rust 和 Bevy 游戏引擎制作的美化版打砖块游戏，具有丰富的视觉效果和流畅的游戏体验。

## ✨ 游戏特色

### 🎨 视觉效果
- **渐变背景动画** - 动态变化的背景色彩
- **粒子爆炸效果** - 砖块被击中时的华丽粒子效果
- **发光效果** - 挡板和球都有美丽的发光边缘
- **彩虹砖块** - 不同行的砖块有不同的颜色主题
- **动态UI** - 根据游戏状态变化的彩色文字
- **脉冲动画** - 挡板发光效果的脉冲动画

### 🎮 游戏玩法
- **经典打砖块** - 用挡板反弹球来击破所有砖块
- **智能反弹** - 球的反弹角度根据击中挡板的位置而变化
- **分数系统** - 不同行的砖块有不同的分值
- **即时重启** - 按R键快速重新开始游戏

### 🎯 控制方式
- **← →** 方向键 - 移动挡板
- **R** 键 - 重新开始游戏（游戏结束后）
- **ESC** 键 - 退出游戏
- **G** 键 - 测试游戏结束（调试用）
- **W** 键 - 测试胜利（调试用）

## 🚀 如何运行

确保你已经安装了 Rust，然后运行：

```bash
cargo run
```

游戏会直接启动，无需选择菜单。
